Lottery Simulator

A realistic and ~99% accurate simulator game. Uses real world statistics to estimate chances of winning with various simulation settings.
Last Updated: CURRENTLY IN DEVELOPMENT

As this game is currently in development, we plan to map out plans for the game on this page.
We plan to support the following lottery games on initial launch: Powerball
Support for Mega Millions and Lucky for Life are planned in a future update.
All lottery games are based off of the NC Education Lottery.

The purpose of this game is to demonstrate the actual chances of you winning a prize in the lottery in the real world.
HOWEVER, please remember that this is a GAME! Do NOT go out and buy lottery tickets after playing this game, especially if you happened to win the in-game jackpot.
This game is also intended to be an alternative to individuals addicted to the lottery, as this game costs nothing to play.

This game will be avaliable to play directly in your web browser.
